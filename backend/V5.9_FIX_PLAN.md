## 🔍 根因分析

### 问题根源
**Commit 499b7ca** - DTO验证错误消息中文化

```diff
- @Length(1, 100, { message: 'Title must be between 1 and 100 characters' })
+ @Length(1, 100, { message: '任务标题必须在1到100个字符之间' })
```

### 影响范围
1. **任务模块**（CreateTaskDto）- 已中文化
2. **认证模块**（LoginDto等）- 部分中文化
3. **其他模块** - 未中文化

### 失败模式
1. **错误消息格式不匹配**（主要）
   - 测试预期英文，实际返回中文
   - 导致断言失败

2. **HTTP状态码不一致**（次要）
   - 预期400，实际500
   - 参数验证逻辑问题
