# Agent任务管理系统

## 项目概述

这是一个基于AI Agent的任务管理系统，支持多Agent协作完成复杂任务。

## 技术栈

### 后端
- NestJS
- PostgreSQL
- Redis

### 前端
- React
- TypeScript

## 项目结构

```

```
